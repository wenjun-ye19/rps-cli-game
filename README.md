# 🎮 RPS CLI Game

A classic **Rock-Paper-Scissors** game built with Python, designed for the command line. This project focuses on clean code structure, user experience (UX), and fundamental logic implementation.

## 🚀 Features

- **Real-Time Countdown:** Immersive 3-second timer before each round using Python's `time` module.
- **Robust Input Handling:** Case-insensitive inputs and validation to prevent crashes.
- **Modular Design:** Separated logic (`is_win`, `play`) from the main execution loop for readability.
- **Interactive Menu:** Easy navigation to Play, View Help, or Quit.
